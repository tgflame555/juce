# Таплинк-сайт юриста

Статический одностраничный сайт для юриста, оказывающего консультационные услуги. Включает:

- первый экран с позиционированием и быстрыми контактами;
- 8 самых запрашиваемых юридических услуг;
- блок "Немного о себе" с квадратами-заглушками под будущие фото;
- всплывающий прайс-лист с оформлением заказа и платежным блоком;
- форму заявки на консультацию;
- разделы об обработке персональных данных и конфиденциальности;
- FAQ, процесс консультации и адаптивную верстку.

## Запуск

Откройте `index.html` в браузере или запустите любой статический сервер в корне проекта.
